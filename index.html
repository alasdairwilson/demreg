DEMREG

Calculate the Differential Emission Measure (DEM) from solar data using regularised inversion.

Introduction.

This code provides a way of recovering the Differential Emission Measure or DEM from temperature sensitive, optically thin, emission from the Sun. This can be from SDO/AIA, Hinode/EIS, Hinode/XRT etc.

There will be more info to follow....

At the moment there are two versions of the code available:

  • DEM Reg - This is the original code that does a single data set
  • DEM Reg Map - This is the newest code that can handle either a single data set, or map. Note that this is a work in progress version of the previous code which only handled SDO/AIA maps. This new version can handle any data type if the appropriate temperature response is given.
